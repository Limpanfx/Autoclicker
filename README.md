# Limpan's Autoclicker

Simple Windows autoclicker with GUI, delay control, keybind toggle, and optional multi-position clicking.

## Usage

1. Run: `install_deps.bat`
2. Run: `runner.bat`
   
- Simple tab: set delay, set keybind, start.
- Positions tab: add screen positions, set delay, set keybind, start.

## Files

- `autoclicker.py` – app
- `install_deps.bat` – deps installer
- `runner.bat` – launcher
